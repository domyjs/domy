# Get Started

Simply create a html file and fill it with this content:

```html
<html>
  <head>
    <title>Domy is awesome !</title>
    <script src="https://cdn.jsdelivr.net/npm/@yoannchb-pro/domy@1.x.x/dist/index.js"></script>
  </head>
  <body>
    <p>Count: {{ this.count }}</p>
    <button @click="this.count++">Increment</button>
    <button @click="this.count--">Decrement</button>
  </body>
  <script>
    DOMY.createApp({
      $watch: {
        count() {
          this.count = Math.max(0, this.count);
        }
      },
      $state: {
        count: 0
      }
    });
  </script>
</html>
```

Open your file in the browser and see the following result !
